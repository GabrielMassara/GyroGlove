# GyroGlove
GyroGlove é uma luva VR de captura de movimento que combina giroscópio e potenciômetros para rastrear, em tempo real, a orientação da mão e a flexão dos dedos. O resultado é um controle mais natural e preciso para interação em realidade virtual, com foco em baixo custo, fácil calibração e integração com microcontroladores.
